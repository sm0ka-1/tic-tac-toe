Tic Tac Toe - PyQt6 Version

A modern Tic Tac Toe game implemented in PyQt6 with score tracking, dynamic turn management, and responsive UI.

Features:
- Responsive board that resizes with the window.
- Score tracking for Player X and Player O.
- Buttons to start a new round or reset scores.

Installation:
- Make sure you have Python 3.10+ installed.
- Install PyQt6: pip install PyQt6
- Run the game: python main.py

How to Play:
- Players take turns clicking empty cells to place their mark.
- The info label shows who starts, whose turn it is, who wins, or if there’s a tie.
- Use Play Again to start a new round with the opposite starting player.
- Use Restart Score to reset both players’ scores.
